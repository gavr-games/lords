Здесь хранятся в папках клиенты игры для раздичных модов.
Например:
mod1/
mod2/
mod3/
...

Каждая папка мода хранит статические массивы js для мода (карты, юниты, здания ...) static_arrays.js,
Шаблон верстки сайта - template.html с маркерами.
Папки для внешнего вида мода.

также в корне есть файлы 
prepare_game.php, который проводит инициализацию игры.
game.php, который подключает статические массивы js для мода, генерирует массивы js для игры, выводит соотв. шаблон.
